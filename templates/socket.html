This module provides access to the BSD socket interface. It is available on all modern Unix systems, Windows, MacOS, and probably additional platforms. The Python interface is a straightforward transliteration of the Unix system call and library interface for sockets to Pythonâ€™s object-oriented
style: the socket() function returns a socket object whose methods implement the various socket system calls. Parameter types are somewhat
higher-level than in the C interface: as with read() and write() operations on Python files, buffer
allocation on receive operations is automatic, and buffer length is implicit on send operations.

Before we start to write some basic Server/Client script lets take a quick look at some of the functions available in Socket module:

  1. socket() -- create a new socket object
  2. socketpair() -- create a pair of new socket objects
  3. fromfd() -- create a socket object from an open file descriptor
  4. fromshare() -- create a socket object from data received from socket.share
  5. gethostname() -- return the current hostname
  6. gethostname() -- return the current hostname
  7. gethostbyname() -- map a hostname to its IP number
  8. gethostbyaddr() -- map an IP number or hostname to DNS info
  9. getservbyname() -- map a service name and a protocol name to a port number
  10. getprotobyname() -- map a protocol name (e.g. 'tcp') to a number
  11. ntohs(), ntohl() -- convert 16, 32 bit int from network to host byte order
  12. htons(), htonl() -- convert 16, 32 bit int from host to network byte order
  13. net_aton() -- convert IP addr string (123.45.67.89) to 32-bit packed format
  14. net_ntoa() -- convert 32-bit packed format IP to string (123.45.67.89)
  15. socket.getdefaulttimeout() -- get the default timeout value
  16. socket.setdefaulttimeout() -- set the default timeout value
