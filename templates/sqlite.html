CRUD

Whats CRUD? CRUD means Create, Read, Update, Delete, and it can mean different things in different systems, But in SQLite (SQL family) it is commonly considered four main operations on table records.

So naturally if we want to corporate any SQL database with in our codes, where it is SQL, MySql, PostgreSQL or in our case SQLite we should be able to understand and execute basic database operations

C for create.

First of all as we are using SQLite3 which is part of the Python standard library we need to Import the module and create a connection Object file.(connection object which is basically database representation in stand alone file.)

Next we call cursor method.(special method to execute SQL commands)

Code:

import sqlite3

connection = sqlite3.connect("db")

cursor = connection.cursor()

now we have connection object and cursor method which is kind a sqlite3 command line, we can create and manipulate data in our database

Table and records.

Table is where we define the structure of database, so basically we are declaring variables and datatypes. for example: we wanna have contacts manager then at least we need to have two variables Name and Phone, lets create a table for these two variables in sqlite3.

---------- Create table ----------

cursor.execute("""CREATE TABLE contacts (Name Text, phone TEXT)***)

Lets take a look at data types in Sqlite3

1 NULL The value is a NULL value.
2 INTEGER The value is a signed integer, stored in 1, 2, 3, 4, 6, or 8 bytes depending on the magnitude of the value.
3 REAL The value is a floating point value, stored as an 8-byte IEEE floating point number.
4 TEXT The value is a text string, stored using the database encoding (UTF-8, UTF-16BE or UTF-16LE)
5 BLOB The value is a blob of data, stored exactly as it was input.
