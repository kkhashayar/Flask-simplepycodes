Lets improve our gui by changing the title of main window and create a Label to display a text. First we create an object from tkinter Label and specify the position of the label which is on the main window then place it on the window by using pack() method.



By creating main_window we have only one frame or window to work with, but imagine we can split the main window by invisible line in to the two separate frames.
By using Frame() method in tkinter we can create as many as frames we want on main window, and as other widgets we have to place them on the main window or root or master screen and basically we can gain more control over our gui.
