But messagebox is not the only method to display information in tkinter, we can even do it by changing labels text while program is running. lets create simple convertor that convert kilometer to miles and instead of display information with messagebox do it with label. because maybe sometimes we dont need to have extra window openning over and over to display some message.
Display mesaage on Label with StringVar
In below example we create very simple converter program which get the kilometer and return miles. StringVar can be changed on run time, so we create a label and instead of text we set the textvariable to value which is already set to StringVar, and in calculate method change it to miles, so by this way we can show the result of calculation on label.
